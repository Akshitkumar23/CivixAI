# **App Name**: CivixAI

## Core Features:

- User Profile Form: Collect and validate user data (age, gender, income, etc.) to create a structured profile.
- Scheme Data Ingestion: Load scheme data (name, eligibility, benefits, documents, etc.) from in-app JSON.
- Eligibility Check: Match user profile against scheme rules using rule-based logic to determine eligibility.
- Benefit Estimation: Estimate potential benefits for eligible schemes using conditional logic.
- Priority Ranking: Rank eligible schemes based on benefit, eligibility ease, documents, and confidence.
- Recommendation Display: Display top 3-5 schemes with benefit estimate, priority, documents, and a 'tool' to determine application process details.
- Chatbot Assistance: Offer basic support in Hinglish
